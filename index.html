Welcome to Trade Route Tracker

Follow these steps to connect your data and start tracking. You only need to do this once.

1
Create a GitHub account
Go to github.com/join and sign up for a free account if you don't have one.
2
Create a personal access token
Go to github.com/settings/tokens/new. Give it any name, set expiration to No expiration, check the repo checkbox, then click Generate token. Copy the ghp_... token — you only see it once.
3
Connect the app
Click ⚙ Settings in the top right. Enter the GitHub username and repository name that the app owner shared with you, paste your token, and click Connect.
4
Add your villages
Use the + Add village button on each page to add your resource villages and army villages. All data syncs automatically to GitHub.
GitHub sync
Data is saved as data.json in the shared repo. Each person needs their own token from github.com/settings/tokens/new — select repo scope.
